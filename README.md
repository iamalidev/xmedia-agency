# XMedia Agency

A modern, beautiful landing page for a digital marketing agency built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Hero** — Full-screen hero with animated headline and CTAs
- **Results** — Animated statistics (120+ projects, 48M+ reach, etc.)
- **Projects** — Interactive iPhone mockup showcasing client Instagram profiles
- **Why Us** — Six key differentiators with testimonial
- **Team** — Team member cards with hover effects
- **FAQ** — Accordion-style frequently asked questions
- **CTA** — Dark call-to-action section
- **Footer** — Contact info, links, newsletter signup, and social media

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- [Next.js 15](https://nextjs.org/)
- [React 19](https://react.dev/)
- [Tailwind CSS 3](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide Icons](https://lucide.dev/)

## Customization

Replace placeholder content in the `components/` directory:
- Update team members in `Team.tsx`
- Modify project showcases in `Projects.tsx`
- Change contact details in `Footer.tsx` and `CTA.tsx`
- Adjust stats in `Results.tsx`
